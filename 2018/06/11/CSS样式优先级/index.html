发表于 2018-06-11

CSS权重

css样式存在权重的概念,同一个标签使用不同的样式会有一个生效一个不生效的问题.

规则:当权重相同的时候权重大的生效,当权重不同的时候后设置的生效.

权重的计算

  • 值左侧的最大,右侧的最小,数位之间没有进位.
  • 0,0,1,0 大于 0,0,0,12

行内样式,外部样式和内部样式的优先级

  • 同一权重下:行内>内部>外部

  • 不同权重下:行内> 权重高的>权重低的
